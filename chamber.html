Chamber

Habilidades

Caçador de Cabeças

Ative para equipar uma pistola pesada.
Use o DISPARO ALTERNATIVO com a pistola equipada para mirar.


Rendezvous

Equipe uma âncora de teleporte.
DISPARE para posicioná-la no chão.
Enquanto estiver no chão e dentro do alcance da âncora,
REATIVE para teleportar rapidamente até a âncora.
A âncora pode ser recuperada para ser REPOSICIONADA.


Marca Registrada

Equipe uma armadilha que busca por inimigos.
DISPARE para posicioná-la no chão.
Quando um inimigo visível entra no alcance, a armadilha inicia
uma contagem regressiva e desestabiliza o terreno ao redor,
criando um campo duradouro que Desacelera os jogadores dentro dele.
A armadilha pode ser recuperada para ser REPOSICIONADA.


Tour de Force

Ative para invocar um poderoso fuzil de precisão personalizado que
abate um inimigo com qualquer acerto na parte superior do corpo.
Use o DISPARO ALTERNATIVO para mirar.
Abater um inimigo cria um campo duradouro que Desacelera os jogadores dentro dele.
